    • 蛋白质搜索引擎 | 质谱仪数据鉴定软件 源代码 源代码 源代码

      2014.02 - 2015.08
      简介：独立设计和实现一款蛋白质搜索引擎的核心算法与功能。

      1. 设计海量浮点数外排算法（500M 内存限制）。为了加速，自行设计分块二进制文件 I/O 方法、内部双精度浮点数基数排序算法、及Str2Dou()/Dou2Str() 取代 STL 函数。利用败者树数据结构实现 k–路归并排序算法，使用多线程技术进行读-排-写。最后，单机上 2.5 亿条记录仅需 250s 即可完成。

      2. 自行设计分词算法精确产生词条集，使用哥德尔编码进行词条去冗余加速，并实现词条-段落-文档的多层倒排索引。最后，以恰当的方式实现查询模块，并设计和实现一个非线性的词条打分函数对匹配相似度进行评估。

      项目结果: 软件性能远超其它引擎，速度平均提高 500%，精度提高300%。

      关键技术: 外排 倒排索引 打分函数 k-路归并 败者树 哥德尔编码 浮点数基数排序 排列组合 C++ 6000行

    • 质谱仪数据解析和定量软件开发 源代码 源代码

      2015.02 - 2016.02
      简介：国家 973 计划项目子课题，独立为搜索引擎在前端和后端流程提供软件支持。

      1. 利用 ILSPY 反编译官方数据解析软件，获得 API 访问权限。根据数据特点，自行设计若干种数据中心化算法和基于信号强度确定噪音基线的去噪方法。同时合理减少 I/O 数提高数据解析速度。

      2. 恰当使用哈希对定量结果计算的过程进行提速，利用方差稳定归一化方法解决异方差分布现象，并设计 PIF 参数量化因信号混杂而对定量结果带来干扰的程度。并使用倒排索引进行词项到文档的定量结果归并。

      项目结果: WiffReader 的数据解析速度比其它软件快 60 倍的同时解析精度却高出 40%。pIsobariQ 的数据定量精度与其它软件一致的同时速度快 90 倍。目前它们均已投入实际使用。

      关键技术: 数据中心化处理 信号去噪 倒排索引 数据解析 定量分析 反编译 Boost C# C++ 7300行

    • 图像搜索引擎 | 数字图像鉴定软件 源代码

      2012.05 - 2013.10
      简介：国家级大学生创新创业训练计划，项目负责人，群体算法在图像检索系统上的应用。

      1. 使用 SOIL 库读取和解析图像库，负责实现 RGB2HSV() 转换颜色空间，自行设计从 HSV 颜色模型中提取颜色直方图、颜色矩和颜色熵等特征方法，构建图像特征库和倒排索引表。通过颜色特征向量量化、单位化及加权平均获得综合特征向量。

      2. 负责利用蜂群觅食时的全局和局部搜索策略寻找全局最优匹配。全局精英保留策略用以加快搜索收敛速度，领域搜索依赖局部最优匹配的策略用以在欧式解空间中寻找更好的匹配。最后，利用 RP-曲线作为评价指标之一。

      项目结果: 较常规搜索，速度提升30倍，精度提高30%。成果已论文发表在 J.Inf.Comput.Sci. 期刊上。

      关键技术: 蜂群算法 特征提取 领域搜索 倒排索引 RP曲线 SOIL库 C++ 6200行

    • 基于 VSM 的引文相似度计算及网络划分软件 源代码

      2015.12 - 2016.03
      简介：引文间的相似度计算和引用关系分析，项目成员

      1. 从引文数据库提取引文题目、摘要及关键词，利用 Boost::tokenizer 进行分词和词干还原处理，词项小写化后去除停用词。构建语料库计算词项的 TF-IDF 并根据位置信息对其进行加权计算，然后取 TF-IDF Top 20 的词项作为每篇引文的特征向量。

      2. 构建倒排索引，计算耦合、共引、直接和间接引用网络，并在向量空间模型下计算引文间的相似度。

      项目结果: 基于上述算法，引文网络划分效果提升20%，成果以论文形式发表在 Lib.Inf.Serv. 期刊上。

      关键技术: TF-IDF 特征提取 加权策略 分词 词干还原 倒排索引 Boost C++ 3200行
