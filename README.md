## Usage

1. Clone 到本地

3. 在 `index.html` 文件中填写你的信息。

4. 生成PDF（以 Win 操作系统 Chrome 浏览器为例）：打开`文件->打印`，设置布局为纵向，纸张尺寸为A3，边距无，背景图片选项打钩，保存。

5. 部署上线。

## Development

```C++
$ npm install
$ npm install -g grunt
$ grunt
```
